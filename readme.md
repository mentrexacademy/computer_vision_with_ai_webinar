# AI Vision Assistant

## Libraries

- streamlit
- opencv-python
- mediapipe
- numpy
- Pillow
- groq
- python-dotenv

## Setup & Installation

```bash
python3 -m venv venv
```

- For Ubuntu/Mac

```bash
source venv/bin/activate
```

- For Macbook with M Series Chip
```bash
export PYTORCH_ENABLE_MPS_FALLBACK=1
```

- For Windows

```bash
.venv\Scripts\activate
```

```bash
pip install -r requirements.txt
```


## Run

```bash
streamlit run app.py
```
